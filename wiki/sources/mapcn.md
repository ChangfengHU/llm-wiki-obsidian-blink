# mapcn

## 核心定位
面向 React 的 shadcn/ui 风格地图组件库，基于 MapLibre GL，支持 Tailwind 和复制粘贴式集成。

- 资料类型：原始项目卡片/分析材料，来源文件：[[raw/mapcn.md]]。
- 关联实体：[[wiki/entities/mapcn|mapcn]]。

- 在知识图谱中用于支撑工具定位、场景边界、风险和横向比较。
